Quick Start with CDN

Add the following script and class to any button to give it the "moving" effect.

<script src="https://cdn.jsdelivr.net/gh/iamovi/button-will-react@cdn/dist/min/v1.0.0/move.min.js"></script> <button class="button_will_react">Click Me!</button>
